## ðŸš€ Getting Started

### Prerequisites
- Node.js (v14+ recommended)
- npm or yarn

### Installation

1. Clone the repository

   ```bash
   git clone https://github.com/raj140302/portfolio.git
   ```

2. Navigate to project directory

   ```bash
   cd portfolio
   ```

3. Install dependencies

   ```bash
   npm install
   # or
   yarn install
   ```

4. Start development server

   ```bash
   npm run dev
   # or
   yarn dev
   ```
